# 🎮 A2R WarZone

A2R WarZone is a competitive online gaming tournament platform 
built for PUBG Mobile and Free Fire players. 

## Features
- 🏆 Tournament registration & management
- 👥 Team formation & profiles  
- 📢 Latest gaming news & updates
- 📍 Based in Bhopal, MP, India

## Tech Stack
- HTML5, CSS3, JavaScript
- Responsive Design (Mobile Friendly)
